Superpowers plugin upload folder (from https://github.com/obra/superpowers )

Use this folder or the matching .zip in the parent directory with your app's plugin / local install flow.

Official install options (Claude Code):
  /plugin install superpowers@claude-plugins-official
  or marketplace: /plugin marketplace add obra/superpowers-marketplace then /plugin install superpowers@superpowers-marketplace

This folder mirrors the repo layout: .claude-plugin, commands, agents, hooks, skills, etc.
